## File a bug

- Use the bug template and attach a repro link from the test page: https://github.com/Simon-He95/markstream-vue/issues/new?template=bug_report.yml
- Include the exact Markdown/content and versions (markstream-vue, Vue, browser/runtime).

## Feature requests

- Use the feature template and describe the use case (AI/chat/SSE, docs, diff viewer, custom nodes): https://github.com/Simon-He95/markstream-vue/issues/new?template=feature_request.yml
