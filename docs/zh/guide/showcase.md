# Showcase

此页面为中文占位，原文（English）：/guide/showcase。

1.0 宣发时建议优先使用：

- [Playground](https://markstream-vue.simonhe.me/)：展示 AI/chat 流式输出、打字机节奏和反向聊天滚动。
- [Test Lab](https://markstream-vue.simonhe.me/test)：展示可分享复现、render mode、diff、thinking、自定义组件、标注和 PDF 导出。
- [Diff sample](https://markstream-vue.simonhe.me/test?sample=diff)、[Thinking sample](https://markstream-vue.simonhe.me/test?sample=thinking)、[Baseline diagrams](https://markstream-vue.simonhe.me/test?sample=baseline)：直接打开对应样例。
- [1.0 Benchmark 报告](/zh/guide/benchmark-1-0)：展示可复跑的性能证据。

跨框架实验页可以作为 labs 展示，但 `markstream-vue@1.0` 的稳定范围仍是 Vue 3 renderer。
